.. _glossary:

Sözlük
======

.. glossary::

    başlık dosyaları
        Başlık dosyaları derlemeli diller için yazılmış olan modül dosyalarıdır.

    manual sayfaları
        (İng. man-pages) Linux standartlarına göre yazılmış, komutlar hakkında detaylı bilgi ve kullanım talimatlarını içeren \
        sayfalardır. `man` komutu ile veya `/usr/share/man` yolundan manual sayfalarına ulaşılabilir.

    belirteç
        tagları belirtmek için kullanılan bazı keywords.

    betik
        çalıştırılabilir dosyalar

    etiket
        Xml dosyası tag

    ikili çalıştırılabilir dosyalar
        derlenmiş

    kütüphane dosyaları
        .h .hxx gibi C ve diğer derlemeli dillere ait başlık dosyalarını içerir.

    script based diller
        Bu programlama dilleri kodları derlemeye gerek kalmadan çalıştırmaya imkan sağlar.
